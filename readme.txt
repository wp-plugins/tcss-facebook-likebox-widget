== Description ==

TCSS Facebook Likebox widget can be used in your wordpress site to show fanbox and users will be able to like your facebook page 

== Installation ==

1. Upload the plugin through `Plugins > Add New > Upload` interface or upload `tcss-facebook-likebox-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the widgets section, select the TCSS 'Facebook Likebox' widget and add to one of your widget areas
4. Set the required settings and save your widget
